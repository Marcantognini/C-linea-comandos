Archivos binarios, memoria dinámica y argumentos del main

Aquí se alojan proyectos que reciben una entrada desde linea de comandos y se realiza una gestion dinamica de memomria, manipulacion de archivos binarios y auxiliares, y uso de estructuras de datos.

Proyectos Incluidos:

1. Buscador de Productos y Ventas
Archivo: `linea_comandos_3.c`
Función:Recibe un código de producto por consola, busca su información y calcula la recaudación total basada en el historial de ventas.

2. Analizador de Beneficios por Sucursal
-Archivo: `linea_comandos_sucursales.c`
-Función:Compara archivos de facturación y costos para identificar sucursales que no cumplen con un piso de beneficio ingresado por el usuario.
